Hello, GitHub Pages!

这是一个通过GitHub Pages搭建的静态网页，无需配置复杂的服务器，只需简单的HTML和CSS代码，就能实现一个美观且可交互的页面。

你可以轻松扩展这个页面的功能，以下是一些常见的扩展方向：

  • 多页面导航：上方导航栏已实现基础跳转功能，只需创建对应的HTML文件（如about.html），即可实现多页面联动。
  • 交互按钮：点击下方按钮体验简单的交互效果（基于JavaScript）：
  • 响应式设计：页面会自动适配手机、平板等设备，在不同屏幕尺寸下保持良好的显示效果。
  • 样式自定义：通过修改CSS代码，可以轻松更换颜色、字体和布局，打造个性化页面。

所有这些功能都无需担心服务器维护，GitHub Pages会自动帮你托管和发布，访问地址为：https://你的用户名.github.io。
